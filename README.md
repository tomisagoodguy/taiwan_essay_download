

# 臺灣博碩士論文網自動下載器

本專案是一個使用 Python 搭配 Selenium 和 ddddocr 開發的自動化工具，旨在協助使用者從「臺灣博碩士論文資訊網」根據關鍵字搜尋並自動下載論文。

腳本會自動處理分頁、點擊、驗證碼辨識、下載、解壓縮及重新命名檔案，並生成一份 `README.md` 來記錄所有已下載的論文，方便管理。

## 主要功能

* **自動化流程** ：自動執行關鍵字搜尋、翻頁、下載論文的完整流程。
* **手動登入** ：程式會先開啟登入頁面，等待使用者手動登入後，再接管後續操作。
* **智慧等待與延遲** ：在操作之間加入隨機延遲，模擬真人行為，降低被網站封鎖的風險。
* **驗證碼自動辨識** ：整合 `ddddocr` 套件，自動辨識下載前的圖形驗證碼。
* **進度記錄與續爬** ：會記錄上次爬取的頁數，中斷後可從上次的進度繼續執行。
* **日誌與索引** ：自動將下載成功的論文連結與資訊記錄在 `README.md` 檔案中，建立一份清晰的索引。
* **檔案整理** ：下載完成後，會自動解壓縮 `.zip` 檔並將論文重新命名為標題，方便管理。

## 使用教學

1. **環境設定** ：

* 確認已安裝 Python 3.10 或更新版本。
* 安裝必要的套件：
  **Bash**

  ```
  pip install selenium ddddocr pillow webdriver-manager
  ```

1. **參數配置** ：

* 打開 `download.ipynb` (或轉換後的 `.py` 檔)。
* 找到檔案最下方的 `if __name__ == "__main__:"` 區塊。
* **修改 `SEARCH_KEYWORD`** ：將 `"台股"` 更改為您想搜尋的論文關鍵字。
* **修改 `DOWNLOAD_LIMIT`** ：設定單次執行的下載數量上限。

   **Python**

```
   if __name__ == "__main__":
       # 1. 設定您要搜尋的關鍵字
       SEARCH_KEYWORD = "人工智慧" 

       # 2. 設定單次執行最多下載幾篇 (建議 200 以下)
       DOWNLOAD_LIMIT = 200 

       # ... (其他延遲設定可視情況調整)

       downloader = ThesisDownloaderWithReadme(
           keyword=SEARCH_KEYWORD,
           max_downloads_per_session=DOWNLOAD_LIMIT,
           # ...
       )
       downloader.run()
```

1. **執行程式** ：

* 儲存修改後，執行此 Python 腳本。
* 程式會自動打開 Chrome 瀏覽器並前往論文網登入頁。
* **請您手動輸入帳號密碼完成登入** 。
* 登入成功後，程式會自動接手，開始搜尋並下載相關文章。
